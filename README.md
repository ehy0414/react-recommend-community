## 프로젝트 소개

---

### 🌿 **프로젝트 소개 - 계절따라** 🌿

**"계절을 따라 떠나는 감성 여행"**

**“계절따라”는** 사용자가 **각 계절에 어울리는 여행지**를 추천받고, 공유할 수 있는 커뮤니티 기반의 여행 플랫폼입니다. 봄, 여름, 가을, 겨울마다 변화하는 다양한 카테고리와 분위기에 맞춰 **감성적인 여행지를 탐색**하고, 직접 경험한 후기를 나누며 여행의 즐거움을 더할 수 있습니다.
